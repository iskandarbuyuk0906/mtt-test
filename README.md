# Didaktika Fanidan Murakkab Testlar Platformasi 🚀

Ushbu loyiha didaktikaning asosiy kategoriyalari va maktabgacha ta’lim didaktikasi bo‘yicha professional testlarni topshirishga mo‘ljallangan zamonaviy, to‘liq ekranli (full-screen) interaktiv veb-platformadir.

Platforma foydalanuvchiga qulay va yuqori darajadagi qulaylik (UX/UI) hissini berish uchun maxsus ishlab chiqilgan.

## ✨ Xususiyatlari

*   **35 ta Murakkab Test Savollari:** Didaktika nazariyasi va maktabgacha ta’lim pedagogikasiga oid mukammal tuzilgan savollar bazasi.
*   **Navigatsiya Tizimi (Oldingi/Keyingi savol):** Foydalanuvchilar istalgan vaqtda "Oldingi savol" tugmasi orqali avvalgi savollarga qaytib, o‘z javoblarini ko‘rishlari mumkin.
*   **"Tasdiqlash" (Apply) Funksiyasi:** Javob variantini bosganda tizim uni darhol tekshirmaydi. Foydalanuvchi fikrini o‘zgartirishi mumkin va faqat "Tasdiqlash" tugmasi bosilgandan keyingina javob qabul qilinadi.
*   **Tezkor va Vizual Qayta Aloqa:** Tasdiqlash tugmasi bosilgach, to‘g‘ri javob avtomatik ravishda **70% shaffoflikdagi yashil rangda**, agar xato belgilangan bo‘lsa, foydalanuvchi tanlovi **70% shaffoflikdagi qizil rangda** porlaydi.
*   **To‘liq Ekranli va Markazlashgan Dizayn:** Chalg‘ituvchi ortiqcha elementlarsiz, faqat testga fokuslanishni ta’minlovchi zamonaviy dizayn tizimi (Soft shadows, Clean Typography).
*   **Yakuniy Hisobot:** Test yakunlangach, barcha savollar bo‘yicha aniq ko‘rsatkich chiroyli panolda aks etadi va testni qayta boshlash imkoniyati beriladi.

## 🛠 Texnologiyalar

Loyiha hech qanday qo‘shimcha kutubxonalarsiz (No frameworks), sof veb-texnologiyalarda yozilgan:
*   **HTML5** - Kontent va struktura
*   **CSS3** - Zamonaviy elementlar, Flexbox maketi va interaktiv render
*   **JavaScript (ES6+)** - Dinamik test mantig‘i va holat (state) boshqaruvi

## 🚀 Ishga tushirish

Loyihani kompyuteringizda yoki internetda ishga tushirish juda oddiy:

1.  Ushbu repozitoriyani yuklab oling (`Clone` yoki `Download ZIP`).
2.  Papka ichidagi `index.html` faylini istalgan brauzerda (Chrome, Safari, Edge, Firefox) ikki marta bosib oching.
3.  Hech qanday server yoki o‘rnatishlar (installation) talab qilinmaydi!
